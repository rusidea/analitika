===== Общая схема работы =====

==== Регистрация случая ====

  - В журнале создаётся учётная карточка, где сохраняется информация о семье, пострадавшей от действий органов системы ювенальной юстиции. Учётная карточка служит средством накопления данных (в том числе документов, фото, видео).
  - Каталогизация, индексация и поиск учётных карточек осуществляются встроенными средствами журнала.

==== Создание статистической формы ====

  - Кроме учётной карточки в журнале создаются формы, в которые вносятся данные для последующей статистической обработки. Такие данные собираются на основе анкетных (опросных) листов, специально разработанных для исследования и мониторинга работы системы ювенальной юстиции.
  - Статистическая обработка данных, полученных путём заполнения форм, осуществляется специально разработанными для этого алгоритмами. Дополнительная статистическая обработка может быть также реализована специализированными программными инструментами.

==== Создание публичной страницы ====

  - Для обнародования фактов работы органов ювенальной юстиции в журнале создаётся публичная страница с описанием пострадавшей семьи, включая те материалы, которые размещены в открытом доступе в сети интернет с согласия членов семьи или их представителей.

===== Инструкции =====

Для создания и поиска учётных карточек и для просмотра публичных страниц используйте кнопки вверху страницы. Дальнейшие инструкции Вы сможете прочитать в шаблонах учётной карточки и публичной страницы после их создания.
