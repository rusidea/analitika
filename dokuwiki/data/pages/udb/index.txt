===== Общая схема работы =====

==== Реализовано ====

  - В базе данных ЮЮ создаётся учётная карточка для семьи пострадавшей от действий органов системы ЮЮ. Учётная карточка служит средством накопления данных о пострадавшей семье.
  - Каталогизация, индексация и поиск учётных карточек осуществляются встроенными средствами журнала "Аналитика".

==== Не реализовано ====

  - Кроме учётной карточки в базе данных ЮЮ также может быть создана форма, в которую вносятся данные для статистической обработки случаев.
  - Статистическая обработка данных формы осуществляется специально разработанными для этого алгоритмами. Дополнительная статистическая обработка может быть также реализована специализированными программными инструментами.
===== Инструкции =====

Для создания и поиска учётных карточек используйте кнопки вверху страницы.
