===== Информационные технологии =====

Информационные технологии, которые определяются в данном стандарте, разработаны в совершенстве и широко используются в сети интернет.

Критериями выбора информационных технологий, используемых для достижения цели данного стандарта, являются:
  * способность сохранять и передавать данные в распределённом виде;
  * возможность верификации данных.

В соответствии с вышеуказанными критериями далее определяются следующие информационные технологии и их назначение для достижения цели данного стандарта.

==== Torrent ====

Полное описание данной технологии содержится в Приложении А.

Технология предназначена для передачи исходных данных в распределённом виде.

Технология позволяет произвести верификацию данных, которые были получены из других источников, а также данных, которые уже хранятся в архиве.

Информационная технология основана на использовании протокола Torrent для передачи данных по сети интернет.


==== Git ====

Полное описание данной технологии содержится в Приложении Б.

Технология предназначена для сохранения, передачи и верификации мета-информации исходных материалов в распределённом виде.

Технология позволяет создать архив исходных материалов.

Распространение = сохранение + передача

Мета-информация - кто, когда, где, что, магнет ссылка

Служебная информация - подпись PGP
==== Media-RSS ====

Полное описание данной технологии содержится в Приложении В.

Технология предназначена для передачи мета-информации исходных материалов в распределённом виде.

Технология позволяет создать дистрибутив мета-информации и канал распространения исходных материалов.
===== Информационная структура =====

Форматы данных:
  - md (мы не можем накладывать ограничения на способ подготовки исходных материалов, поэтому torrent)
  - txt
  - pgp
